## Instructions

1. Clone this repo:
   ```sh
   $> git clone https://github.com/RogWilco/Resume.git
   ```
2. Install dependencies:

   ```sh
   $> npx yarn install
   ```

3. Build:

   ```sh
   $> yarn dist
   ```

4. Access the build output under `out/dist/`.
